Perform Assignment “执行赋值”或“进行赋值”
"The function will perform the assignment of the value to the variable." 可以翻译为：“该函数将执行将值赋给变量的操作。”
"To update the variable, you need to perform an assignment with the new value." 可以翻译为：“要更新变量，你需要用新值进行赋值。”
Translate Assignment “翻译赋值”或“转换赋值”
"The compiler will translate the assignment into machine code." 可以翻译为：“编译器将赋值操作翻译成机器码。”
"In the interpreter, the translate assignment function converts high-level language assignments to intermediate code." 可以翻译为：“在解释器中，翻译赋值函数将高级语言的赋值操作转换为中间代码。”
